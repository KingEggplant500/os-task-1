# os-task-1
Утилита на Bash для обработки аргументов командной строки.

В рамках задания была разработана консольная утилита на языке Bash, которая умеет принимать аргументы командной строки и выполнять три основных действия. Первое действие это вывод списка пользователей и их домашних директорий. Второе действие это вывод списка процессов, отсортированных по идентификатору процесса. Третье действие это вывод справочной информации о доступных аргументах и примеров использования утилиты.

Утилита поддерживает как короткие аргументы u p h так и длинные аргументы users processes help. Также в утилите реализована поддержка перенаправления обычного вывода в файл и перенаправления ошибок в отдельный файл. Для этого используются аргументы log и errors которым указывается путь к файлам. Перед перенаправлением выполняется проверка того что путь существует и доступен для записи.

Каждый тип действия вынесен в отдельную функцию чтобы структура программы была понятной и соответствовала требованиям задания. Для вывода пользователей используется чтение файла etc passwd и форматирование данных. Для вывода процессов используется команда ps с последующей сортировкой по PID. Для проверки корректности путей реализована отдельная функция которая проверяет что директория существует что в неё можно записывать и что файл можно создать.

Разбор аргументов выполнен с использованием механизма getopts который позволяет корректно обрабатывать как короткие так и длинные параметры. Также обработаны ситуации когда пользователь вводит неизвестный аргумент или не указывает никаких действий. В таких случаях выводится понятное сообщение об ошибке и подсказка по использованию утилиты.

Утилита полностью работает на macOS и Linux так как использует только стандартные средства Bash.
